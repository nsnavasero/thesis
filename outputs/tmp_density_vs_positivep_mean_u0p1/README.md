# Temporary Density-Matrix vs Positive-P Comparison, U=0.1

This folder contains a standalone 2-site comparison for:

- initial Fock state `|1,0>`
- `U = 0.1`
- `gamma = 1`
- `J = 1`
- total time `5 s`

The figures use `scienceplots` styling and show the full density-matrix curve together with the Positive-P curve over its numerically stable time window.
